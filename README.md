# 🤖 AI Study Assistant

An AI-powered web application that helps students study smarter using real artificial intelligence.

## 🚀 Features
- ✨ Summarize long content instantly
- ❓ Generate smart study questions
- 👶 Explain concepts in a simple way (ELI5)
- 🎯 Clean and modern UI

## 🧠 Powered By
- OpenRouter API (Free AI models)
- JavaScript (Frontend)

## 📸 How It Works
1. Paste your study content
2. Choose an action (Summarize / Questions / Explain)
3. Get instant AI-powered results

## 🔧 Setup

1. Clone the repo: https://ansosama1.github.io/ai-study-assistant/


2. Open `script.js` and add your API key:

const API_KEY = "your_api_key_here";


3. Open `index.html` in your browser
